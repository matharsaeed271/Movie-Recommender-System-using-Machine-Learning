# Movie-Recommender-System-using-Machine-Learning
#### Now scale and classify your thoughts with the cutting edge web app
This web application recommends movies to users based on their preferences using machine learning techniques. The system analyzes user input and suggests similar movies by comparing features such as genres, keywords, and ratings.

This project demonstrates the practical application of machine learning in personalization systems, commonly used by platforms like Netflix and Amazon.

The web application is deployed online and can be accessed here:

👉 https://recommender-movies.streamlit.app/
